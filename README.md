## 🚀 快速启动

### 方法 1: 使用 Python（推荐）

```bash
# 进入 demo 目录
cd C:\Users\EF\.openclaw\workspace\demo

# 启动本地服务器
python -m http.server 8080

# 浏览器访问
http://localhost:8080
```

### 方法 2: 使用 Node.js

```bash
# 安装 http-server（首次使用）
npm install -g http-server

# 进入 demo 目录并启动
cd C:\Users\EF\.openclaw\workspace\demo
http-server -p 8080

# 浏览器访问
http://localhost:8080
```

### 方法 3: 直接打开

直接双击 `index.html` 文件即可在浏览器中打开（部分功能可能受跨域限制）。
